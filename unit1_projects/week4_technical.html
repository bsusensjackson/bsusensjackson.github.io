Enumerable#map

Enumerable methods are actually modules that are included in the Ruby programming language. These provide for ways for programmers to use the Ruby language without having to rewrite the methods each time they want to use them.

One very useful enumerable method is the collect method, which can be used exactly for what it states - to collect information from a block of code. Running:

>my_array = [1, 2, 3, 4, 5]
>my_array.collect { |x| x+1 }

Will produce,

=> [2, 3, 4, 5, 6]

And #map and #collect are completely interchangeable - they do the same thing.

This is very powerful and saves the programmer a lot of time!
